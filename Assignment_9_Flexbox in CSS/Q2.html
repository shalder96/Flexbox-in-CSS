Properties of Flex-box layout model

The Flexbox layout model in CSS is designed to provide a more efficient way to design complex layouts in a more predictable and consistent manner. Here are explanations for the key properties in the Flexbox layout model:

  1. Justify Content:
    • Definition:This property aligns flex items along the main axis of the flex container.
    • Values:
      • flex-start: Items are packed toward the start of the flex container.
      • flex-end: Items are packed toward the end of the flex container.
      • center:Items are centered along the main axis.
      • space-between: Items are evenly distributed in the container with space between them.
      • space-around: Items are evenly distributed in the container with space around them.
      • space-evenly: Items are evenly distributed in the container with equal space around them.
  2. Align Items:
    • Definition:This property aligns flex items along the cross axis of the flex container.
    • Values:
      • flex-start: Items are aligned to the start of the cross axis.
      • flex-end: Items are aligned to the end of the cross axis.
      • center: Items are centered along the cross axis.
      • baseline Items are aligned to the baseline of the cross axis.
      • stretch: Items are stretched to fill the container along the cross axis.
  3. Gap:
    • Definition: This property sets the gap (space) between flex items in a flex container.
    • Values:
      • A length value (e.g., 10px, 2rem) to specify a fixed gap.
      • A percentage value (e.g., 5%) to specify a percentage of the container's size.
      • normal: Default value, equivalent to about 1em.
  4. Flex Direction:
    • Definition:This property defines the direction in which flex items are placed in the flex container.
    • Values:
      • row: Default value. Items are placed in the same direction as the main axis.
      • row-reverse: Items are placed in the opposite direction as the main axis.
      • column: Items are placed in the same direction as the cross axis.
      • column-reverse: Items are placed in the opposite direction as the cross axis.
  5. Flex Wrap:
    • Definition: This property defines whether flex items are forced onto a single line or can be wrapped onto multiple lines.
    • Values:
      • nowrap:Default value. All flex items are on one line.
      • wrap: Flex items are wrapped onto multiple lines if needed.
      • wrap-reverse: Flex items are wrapped onto multiple lines in the reverse order.
